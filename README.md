# Loan Approval Prediction System

## 📌 Overview
This project predicts whether a loan application will be approved or rejected using the Random Forest algorithm. It helps automate decision-making in financial systems.

## 🚀 Features
- Predicts loan approval status
- Data preprocessing and feature engineering
- High accuracy using ensemble learning
- User input-based prediction system

## 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy

## 📂 How to Run
1. Install required libraries:
   pip install pandas numpy scikit-learn
2. Run the model file
3. Enter input values to get prediction
